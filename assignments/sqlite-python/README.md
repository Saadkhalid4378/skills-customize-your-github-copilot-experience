# 📘 Assignment: SQLite with Python

## 🎯 Objective

Learn how to use SQLite with Python to store, query, and manage data persistently. Practice database creation, table operations, and CRUD commands in a simple Python application.

## 📝 Tasks

### 🛠️ Create and Populate a SQLite Database

#### Description

Write a Python program that creates a SQLite database and a table, then inserts several records into it.

#### Requirements
Completed program should:

- Use Python's built-in `sqlite3` module
- Create a SQLite database file and a table with at least three columns
- Insert multiple rows of sample data into the table
- Use parameterized SQL statements to prevent SQL injection
- Display a confirmation message after data insertion

### 🛠️ Query and Modify Database Records

#### Description

Add functions that read, update, and delete records from the database, then show the results.

#### Requirements
Completed program should:

- Query and display all records from the table
- Fetch and display a single record by its ID
- Update one or more fields for an existing record
- Delete a record and confirm the removal
- Print query results in a readable format
