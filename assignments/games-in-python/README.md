
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman game using Python fundamentals. Practice string manipulation, loops, conditionals, and user input while managing game state and player feedback.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description

Write a Python program that selects a random word from a predefined list and lets the player guess letters until the word is solved or guesses run out.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list
- Display the word with underscores for unguessed letters
- Accept single-letter guesses and reveal letters in the word correctly
- Track remaining incorrect guesses and update the player
- End the game with a win message when the word is guessed or a lose message when attempts are exhausted

### 🛠️ Improve Game Interaction

#### Description

Add gameplay interaction features that make the game easier to use and more user-friendly.

#### Requirements
Completed program should:

- Show letters that have already been guessed
- Handle invalid input and repeated guesses gracefully
- Allow the player to restart the game or quit after the game ends
- Keep the gameplay loop clear and easy to follow
